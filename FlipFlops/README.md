# Flip-Flops

This folder contains basic sequential logic flip-flop designs implemented using Verilog HDL.

## Included Designs
- SR Flip-Flop
- JK Flip-Flop
- D Flip-Flop
- T Flip-Flop

## Topics Covered
- Sequential Logic
- State Transition
- Timing Behaviour
- Clocked Circuits
- Excitation Tables

## Tools Used
- Verilog HDL
- ModelSim / Vivado
- Logisim

## Applications
- Counters
- Registers
- Memory Elements
- FSM Design
