# Arithmetic Circuits

## Overview

Arithmetic circuits are the fundamental building blocks of digital systems and VLSI architectures. They perform mathematical operations such as addition, subtraction, multiplication, and accumulation.

These circuits form the core of:

- Arithmetic Logic Units (ALUs)
- Digital Signal Processors (DSPs)
- Microprocessors
- Microcontrollers
- GPUs
- AI Accelerators
- Communication Systems

This repository contains Verilog HDL implementations of commonly used arithmetic circuits along with truth tables, design equations, simulation results, and testbenches.

---

## Circuits Included

| Circuit | Description |
|----------|-------------|
| Half Adder | Adds two single-bit binary numbers |
| Full Adder | Adds three single-bit binary inputs |
| Half Subtractor | Performs single-bit subtraction |
| Full Subtractor | Performs single-bit subtraction with borrow |
| Ripple Carry Adder | Multi-bit adder using cascaded Full Adders |
| Carry Lookahead Adder | High-speed adder using carry generation logic |
| Carry Save Adder | Reduces addition delay in multipliers |
