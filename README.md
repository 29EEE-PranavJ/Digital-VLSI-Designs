# Digital-VLSI-Designs

This repository contains digital design architectures, arithmetic circuits, sequential logic systems, and VLSI-oriented hardware implementations developed as part of my learning journey in digital electronics and semiconductor system design.

## Topics Covered
- Flip-Flops
- Counters
- FSM Design
- Arithmetic Circuits
- Wallace Tree Multipliers
- Compressor-Based Architectures
- Verilog HDL

## Tools Used
- Logisim
- ModelSim
- LTspice
- Verilog HDL

## Goal
To explore high-speed and optimized digital hardware architectures relevant to VLSI systems, processors, DSP, and AI acceleration.
